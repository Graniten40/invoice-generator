# Free Invoice Generator

A simple and free online invoice generator built with React, TypeScript and Vite.

The tool lets users create invoices directly in the browser, calculate VAT automatically, choose between USD and EUR, and download the invoice as a PDF.

## Features

- Create invoices in the browser
- Add company details
- Add client details
- Add invoice number
- Add invoice date and due date
- Prevent due date from being earlier than invoice date
- Add invoice items
- Calculate subtotal, VAT and total automatically
- Choose between USD and EUR
- Save invoice data locally in the browser
- Download invoice as PDF
- No account required
- No backend required

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- jsPDF
- html2canvas
- localStorage
